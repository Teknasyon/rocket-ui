# Checkbox

Checkbox is a component that allows the user to select one or more options from a set. It can be used to select multiple options in a form.

It can also be used as a single option selector, if you want to present the user with only one option to choose from.

The user can select or deselect the option by clicking on the checkbox.

The user can also select or deselect the option by clicking on the label next to the checkbox.

`*`The user can also select or deselect the option by using the keyboard. The user can use the spacebar to select or deselect the option.
